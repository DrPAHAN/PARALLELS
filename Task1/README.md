# Sinus Sum Project
Этот проект заполняет массив из 10^7 элементов значениями синуса (один полный период) и вычисляет их сумму.

## Как собрать
Создайте директорию build и перейдите в неё:
mkdir build
cd build

## Для сборки с double (по умолчанию):
cmake ..
make

## Для сборки с float:
cmake -DUSE_FLOAT=ON ..
make

## Запустить 
./sinus_sum
